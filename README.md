# Developer Portfolio

🔗 **Live Site:** https://cameronmcrae.ca/

A responsive personal portfolio website built with **React** and **React Bootstrap**, designed to showcase projects, skills, and experience in a clean, modern interface.

> This project is actively maintained and evolving.

---

## Overview

This portfolio serves as a central hub for my work as a developer, highlighting:

- Featured projects
- Technical skills
- Experience and background
- Contact information

The goal is to present my work in a way that is both **visually clean** and **easy to navigate**, while remaining performant and scalable.

---

## Tech Stack

### Frontend
- React
- React Bootstrap
- JavaScript (ES6+)
- HTML5 / CSS3

### Tooling
- Git & GitHub
- npm

---

## Features

### Project Showcase
- Highlighted projects with descriptions
- Links to live demos and GitHub repositories
- Expandable for future case studies

### Responsive Design
- Fully responsive layout using React Bootstrap grid system
- Optimized for desktop, tablet, and mobile devices

### Clean UI/UX
- Minimalist design for readability
- Consistent component-based structure

### Contact Section
- Easy way for recruiters or collaborators to reach out
- Expandable for email service integrations (e.g., Formspree, EmailJS)
